## Alias
1. create 
    ```
    alias foo = "cd ..; ls; cd ..; ls"
    ```
2. delete
    ```
    unalias foo
    ```
3. Alias will be gone if the terminal is closed. To make these alias persistent, store them in bashrc/zshrc.
    ```
    nano .zshrc
    ```

## History
1. history of inputs
    ```
    history
    ```

## Details of a command
1. type of a command
    ```
    type ls
    ```
2. manual page of a command
    ```
    man ls
    ```
3. whatis
    ```
    whatis ls
    ```
